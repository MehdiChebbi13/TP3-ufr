Project Algorithms
University of Rouen

Structure:
- src/: Source code
- include/: Header files
- obj/: Object files
- bin/: Executables
- data/: test data
- plots/: performance plots

Compilation:
Run `make` to compile all targets.
Targets:
- genere-texte
- genere-mots
- ac-matrice
- ac-hachage

Cleaning:
Run `make clean` to remove object files and executables.
